# Security Policy

## Supported Versions

At impe-byte, we take the security of our infrastructure and open-source projects seriously. We currently provide security updates for the following versions of FileSpy:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within FileSpy, please DO NOT open a public issue. We practice coordinated disclosure to protect our users. 

Please report any security-related issues or potential vulnerabilities directly to our technical team via email. 

**Response Protocol:**
1. We will acknowledge receipt of your vulnerability report within 48 hours.
2. Our security team will investigate and verify the vulnerability.
3. If verified, we will develop a patch and deploy it to the main branch.
4. A public security advisory will be issued only after the patch is confirmed stable and live.

Thank you for contributing to the security of the impe-byte ecosystem.
